## 问题描述

`readKnowledgeDocuments()` 扫描全局知识仓库，解析 frontmatter，提取日期和标题。读取逻辑错误会导致 Board 展示异常或知识文档遗漏。

---

## 测试验证项

### 验证 10.1: 目录不存在返回空数组
- **预期**: `[]`，不抛异常
- **用例**: TC-C-001

### 验证 10.2: 空目录返回空数组
- **预期**: `[]`
- **用例**: TC-C-002

### 验证 10.3: 递归扫描三层结构
- **预期**: 正确解析 project/type/filename
- **用例**: TC-C-003

### 验证 10.4: Frontmatter 完整字段解析
- **字段**: `title`, `date`, `project`, `topic`
- **用例**: TC-C-004

### 验证 10.5: YAML Date 对象 → 字符串
- **预期**: js-yaml 解析的 Date 转为 `YYYY-MM-DD`
- **用例**: TC-C-005

### 验证 10.6: 文件名日期前缀提取
- **输入**: `2026-03-10-ideas.md`
- **预期**: `date="2026-03-10"`, `title="ideas"`
- **用例**: TC-C-007

### 验证 10.7: Frontmatter title 优先级高于文件名
- **输入**: 文件名 `legacy.md`，frontmatter `title: "New"`
- **预期**: `title="New"`
- **用例**: TC-C-009

### 验证 10.8: 按日期降序排序
- **预期**: 新日期在前，无日期在最后
- **用例**: TC-C-012

### 验证 10.9: project/type 过滤
- **预期**: 组合过滤仅返回匹配文档
- **用例**: TC-C-013~015

### 验证 10.10: 忽略隐藏文件/目录
- **预期**: `.hidden.md` 和 `.git/` 被排除
- **用例**: TC-C-016~017

### 验证 10.11: 非 `.md` 文件忽略
- **预期**: `.txt`, `.json` 不纳入
- **用例**: TC-C-020

### 验证 10.12: 损坏/不可读文件静默跳过
- **预期**: 不中断扫描，不抛异常
- **用例**: TC-C-019
